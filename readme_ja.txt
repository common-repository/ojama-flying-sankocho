何の役にも立たないお遊びなプラグイン。シンプルに描かれた三光鳥が出て来て作業の邪魔をするだけです。

== Description ==

三光鳥（日本に居る鳥）がひらひらと出て来て、マウスカーソルにまとわりつきます。

マウスの動きを止めるとここぞとばかりにそのマウスカーソルを乗っ取ってしまいます。

何度かクリックすると、そのうち飛び去っていきます。

ただ、それだけです。ただ、うっとおしいだけです。

三光鳥（Japanese Paradise Flycatcher）というのは夏に日本の深い森で子育てをする鳥で、バードウォッチャーにはとても人気のある鳥です。

その姿も特徴的で愛らしいのですが、鳴き声もまた独特で楽しませてくれます。

それは「ツイ、ヒー、ホイ、ホイホイホイ」という感じで聞こえ、「月、日、星、ホイホイホイ」と聞きなされ、それが名前にある三つの光の由来です。

私の大好きな鳥なので遊びで作っただけのプラグインです。何の役にも立ちません。

詳しい使い方や解説は[作者の解説ページ](http://strix.main.jp/?diys=wordpress-ownmade-calendar)をご覧ください。

== Installation ==

1.　ダウンロードし解凍してできたflying_jpf.phpが入っているフォルダごとプラグインディレクトリ`/wp-content/plugins/`にアップロードします。
2.WordPressのプラグインメニューにおいてプラグインを有効化します。
3.管理画面の設定メニューに入り、各設定をします。
4.初期状態では起動しない設定になっています。
5.タッチパネル、もしくは横解像度が770pxより小さいデバイスによるアクセスの場合は起動しないようにしてあります。

== Frequently asked questions ==

= 対応しているウェブブラウザは何ですか？ =
HTML5とCSS3に対応しているウェブブラウザに対応しています。

= 対応している言語は何ですか？ =
日本語と英語の対応ですが、英語は改善が必要だと思われます。

= プラグインを削除した後、保存されているオプションの値はどうなりますか？ =
WordPressのプラグインメニューからプラグインを削除すれば、保存されているオプション値も消去されます。
